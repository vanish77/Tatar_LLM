## ?? Как использовать окружение

### Вариант 1: Автоматическая активация (рекомендуется)

```bash
source activate_env.sh
```

### Вариант 2: Ручная активация

```bash
source venv/bin/activate
```

### Вариант 3: В новом терминале

```bash
cd "/Users/ivanevgenyevich/Documents/Study/ММОТ_NLP"
source venv/bin/activate
```

## ?? Проверка что окружение активно

После активации вы увидите `(venv)` в начале строки терминала:

```bash
(venv) user@macbook ММОТ_NLP %
```

Также можно проверить:

```bash
which python  # Должен показать путь к venv/bin/python
python --version  # Должен показать Python 3.12.6
```
